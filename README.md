## Usage

### Nest
Call `nest()` inside your project's stylus directory.
```js
nest(1.5, -2.5, 6)
```

### Grid
A grid consists of:
```html
<div class='grid'>
  <div class='grid-items'>
    <!-- Elements to be displayed in a grid -->
  </div>
</div>
```
